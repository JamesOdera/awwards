# <a href="https://aww-1.herokuapp.com/">awwards</a>
an awwards app wher users can post their projects to be reviewed and rated and they can also review other peoples project

## Author: 

# James Odera.

## Decription
This app is created to enable students and other devs post their apps so that they can be reviewed

## Prerequisites
Things you need to install the software.

- python3.6.7

## Things you can do

- Sign in to the application to start using.

- See your profile.

- Edit your Profile incase you need to change it

- see the rating of other projects

- rate other projects 

- hover over the project to see details


## Technologies Used

- python django 

- Javascript

- html css

- bootstrap3
